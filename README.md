# Southside School of Music

An Electron + React desktop app that teaches music theory and creation using embedded YouTube videos, game loops, and ear training lessons across genres like industrial, blues, house, rock, funk, salsa, and metal.

## Features
- Modular theory card system
- Audio and visual training games
- Embedded YouTube listening assignments
- Simple UI for composition and music creation
- Designed for visual and neurodiverse learners
