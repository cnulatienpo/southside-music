# Circuits Visual World System

## Aesthetic
- industrial, urban, 2D
- alley steam vents
- metro platforms
- corner stores
- laundry rooms
- flea market aisles
- rooftops with fans

## How Scenes Render
- `backgroundRegistry` defines spaces
- `backgroundRenderer` provides layers + parallax hints
- `ambientAnimation` adds movement
- `sceneLayoutEngine` positions NPC + text + tools

## Why 2D?
- fast
- cheap
- flexible
- accessible

## Integration
- UI hooks read RenderInstructions
- No correctness, no grading
- Keeps Circuits playful + low-pressure
