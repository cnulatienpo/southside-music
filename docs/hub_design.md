# The HUB — Southside’s Industrial Crossroads

## Purpose
This is the main entry into the whole game world.

## Tone & Vibe
- urban industrial hallway
- neon arrows
- graffiti labels
- crates, cables, old signs
- rooms branching everywhere

## Player Experience
- absolute freedom
- pick ANY mode
- pick ANY topic
- choose a Circuit run
- continue a past run
- wander

## No Grading
- The HUB never locks anything
- Everything is open

## Integration
- CircuitsEngine
- SceneProgressionEngine
- StoryChains
- All game modes
